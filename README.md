# 🎬 Movie Management API
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-blue?style=for-the-badge)

API REST em desenvolvimento para gerenciamento de filmes, categorias e plataformas de streaming, utilizando Spring Boot, autenticação JWT e arquitetura em camadas.

## 🛠️ Tecnologias

- Java 17
- Spring Boot
- Spring Data JPA
- Spring Security
- PostgreSQL
- Auth0 Java JWT
- Docker

## ▶️ Como executar

> Em desenvolvimento. Instruções serão adicionadas em breve.

A aplicação será executada via Docker, sem necessidade de instalar Java ou Maven localmente.

**Pré-requisitos futuros:**
- Docker instalado
- Variáveis de ambiente configuradas (`.env`)
