# 🎬 Movie Management API
API REST para gerenciamento de catalogo de filmes, permitindo cadastro de filmes, organização por categorias e associação com plataformas de streaming, além de implementar autenticação de usuários para acesso seguro.

---
## 📌 Objetivo
Desenvolver uma API robusta e escalável para:
- Cadastro de filmes
- Organização por categorias
- Associação com plataformas de streaming
- Autenticação de usuarios (registro e login com JWT)

---
## 🛠️ Tecnologias
- Java
- Spring Boot
- Spring Data JPA
- Spring Security
